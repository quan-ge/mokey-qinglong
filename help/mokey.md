# 猴子脚本在青龙面板中使用

1.在青龙面板中添加订阅

>
>
>名称：     mokey
>
>订阅链接： https://gitproxy.click/https://github.com/quan-ge/mokey-qinglong.git
>
>分支：     main
>
>白名单：   mokey_ql_*
>
>黑名单：   beta_*
>
>定时：     0 0 1 * *
>


2. 点击确定后 运行一次

3. 回到青龙面板主页，点击更多

4. 创建视图：

>
> 视图名称:
> [mokey]猴子脚本
>
> 筛选条件:
> 订阅 属于 mokey
>

5. 全选，点击全部禁用

6. 您已经完成✅猴子脚本的安装了，

[请您前往脚本使用教程。](https://github.com/quan-ge/mokey-qinglong?tab=readme-ov-file#%E8%84%9A%E6%9C%AC%E4%BD%BF%E7%94%A8%E6%95%99%E7%A8%8B)
