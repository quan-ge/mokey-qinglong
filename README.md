# 猴子脚本
目前，青龙面板脚本网络上已经很多了。

不过，仍然有一部分没有人编写脚本的“空白”。
猴子脚本会填补一小部分“空白”。


“mokey”的名字来源于猴子的英文“monkey”，

但可能给人不稳定不靠谱的感觉，所以删除字母N。
